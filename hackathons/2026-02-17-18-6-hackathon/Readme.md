## What (was it)

This is a Hackathon for the SCS Community. While it is named "Hackathon" all activities around digital sovereign cloud infrastructure in the context of the SCS project are welcome. This ranges from hacking code, creating and improving docs, sharing and fostering knowledge to just having fun with like minded people.

This SCS Hackathon will focus specifically on SCS KaaS (Kubernetes-as-a-Service) standards and certification. If you work with Kubernetes, want to help shape the standards, and aim to achieve certification in the end, this is the right place for you.

_Highlight:_ On Tuesday afternoon, there will be a guided tour of the Nuremberg–Fürth Data Center. If you would like to take part, please indicate this during registration.
--> Important: You must bring a valid ID with you.

For this to be something that everyone values and enjoys we have certain rules which are laid out in the [Code of Conduct](https://docs.scs.community/community/mission-statement#code-of-conduct).

The host of the SCS Hackathon will be Rechenzentrum Nürnberg-Fürth – ScaleUp Technologies.
